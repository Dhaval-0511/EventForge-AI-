# 🚀 EventForge AI
### AI-Powered Event Lifecycle Operating System

EventForge AI is a Generative AI-driven platform that automates and streamlines the complete lifecycle of events for colleges, campus clubs, communities, and organizations.

It unifies event creation, promotion, registration, attendance tracking, certification, documentation, and analytics into one intelligent ecosystem.

---

## 🌟 Vision

To democratize professional-grade event creation and management by empowering every student, club, and organization with accessible AI-powered tools.

---

## 🎯 Problem Statement

Event organization today is fragmented and inefficient. Teams typically rely on multiple disconnected tools:

- Canva → Poster design  
- Google Forms → Registrations  
- Excel → Attendance tracking  
- WhatsApp → Communication  
- Google Docs → Reports  
- Manual workflows → Certificate creation  

This leads to:

- Repetitive manual work  
- Poor coordination across teams  
- Data inconsistency  
- Lack of structured documentation  
- No centralized analytics  
- Operational inefficiency  

There is no unified AI-powered platform that integrates content creation, management, automation, and insights into one seamless system.

---

## 💡 Solution Overview

EventForge AI transforms event management into a structured lifecycle:

Draft → Design → Publish → Execute → Document → Analyze

It eliminates tool fragmentation and introduces automation, AI assistance, and data-driven insights into campus event ecosystems.

# 🧩 Core Features

---

## 🔐 1. Authentication & Organization Management

- Secure JWT-based authentication
- Multi-organization support
- Organization creation & configuration
- Member invitation via email
- Custom role assignment
- Permission-based access control

Users can belong to multiple organizations and collaborate efficiently.

---

## 👥 2. Role-Based Access Control (RBAC)

Predefined roles:

- Admin
- Tech Head
- Marketing Head
- Documentation Lead
- Finance Head

Granular permissions ensure secure and structured operations:

- Create/Edit Events
- View Participants
- Mark Attendance
- Generate Certificates
- Access Analytics
- Manage Reports

The UI dynamically adapts based on user role.

---

## 📅 3. Event Management Engine

Comprehensive event lifecycle management:

- Draft and publish workflow
- Event categories (Workshop, Hackathon, Seminar, etc.)
- Online / Offline / Hybrid mode
- Registration deadline management
- Event templates
- Clone previous events
- State transitions:
  Draft → Published → Ongoing → Completed → Archived

---

## 🧩 4. No-Code Registration System

Built-in customizable registration system:

- Drag-and-drop form builder
- Multiple field types (Text, Email, Dropdown, File Upload, etc.)
- Field validation & required toggles
- Registration approval workflow (optional)
- CSV export
- Unique participant ID generation
- Automated confirmation emails

Participants cannot view other registrants, ensuring privacy.

---

## 📲 5. QR-Based Smart Attendance

- Unique QR code generated per participant
- Real-time QR scanning
- Duplicate scan prevention
- Timestamp logging
- Manual attendance toggle
- Bulk upload support

Attendance data automatically links to certification eligibility.

---

## 🏆 6. Certificate Automation System

- Custom certificate template upload
- Dynamic placeholders:
  - {{name}}
  - {{event_title}}
  - {{date}}
- Bulk PDF generation
- Email-based certificate distribution
- Unique certificate ID for verification
- Public certificate validation link

---

### 📄 AI Documentation Generator
Creates:

## 📊 8. Analytics Dashboard

### Organization-Level Insights
- Total events conducted

Rule-based workflow system:
